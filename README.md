# IPLocator

## Getting Started

### 1. Navigate to the server folder
```bash
cd server
```

```bash
npm install express mysql cors nodemon
```
**Create .env file**

```bash
API_KEY=
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_DATABASE=
```
**Then run**

```bash
npm start
```

### 2. Navigate to the client folder

```bash
cd client
```

```bash
npm install vite --save-dev
```

**Then run**

```bash
npm run dev
```
### 3. Navigate to this url

[http://localhost:5173/login](http://localhost:5173/login)
